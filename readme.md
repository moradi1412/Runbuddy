# RunBuddy

A sample personal trainer website built to showcase responsive layout, web design fundamentals, and interactive page structure.

## Live Site
https://moradi1412.github.io/Runbuddy/

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Overview
RunBuddy is a landing page for a fitness trainer brand, designed with a clean user interface and structured page sections.
The site includes a navigation header, hero section, service highlights, sign-up form, and footer content.

## Skills Demonstrated
- Building a mobile-friendly webpage layout using HTML semantic structure
- Styling page sections, text, and images with CSS selectors, classes, and IDs
- Using relative paths to link HTML files with CSS and assets
- Creating navigation and page sections such as header, footer, and hero content
- Designing interactive forms and sign-up input fields for user engagement
- Applying CSS positioning, float layout techniques, and SVG usage for visual elements
- Implementing responsive design patterns for a polished personal trainer website
- Debugging and refining layout with Chrome DevTools

## Features
- Responsive trainer landing page layout
- Styled sections for services and training offerings
- Contact sign-up form for prospective clients
- Clear visual hierarchy and navigation
- Use of modern web design practices for a small business site
